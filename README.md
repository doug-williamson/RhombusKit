# RhombusKit
Nx + pnpm monorepo for RhombusKit — a framework-agnostic design token system and Angular component library. Ships @rhombuskit/tokens, @rhombuskit/theme-engine, @rhombuskit/material-preset, and @rhombuskit/core.

## Status

RhombusKit is pre-1.0. The API is **not yet stable**.

- Versions in `0.x` may include breaking changes between minor releases.
- Breaking changes are documented in [CHANGELOG.md](./CHANGELOG.md) and called out
  in each release's GitHub release notes.
- The first stable release will be `1.0.0`, at which point semver guarantees take
  effect.

If you're evaluating RhombusKit for production, wait for `1.0.0` or pin to an exact
`0.x.y` version and review the changelog before upgrading.
